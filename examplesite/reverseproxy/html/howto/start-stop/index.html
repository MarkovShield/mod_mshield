Start & Stop Apache Web Server

Please use the start and stop option in the Hacking-Lab menu for starting and stopping your LiveCD web server.

This will start or stop both, the Frontend Apache and Backend Apache web server.

Command Line

Item File
Start Apache Front & Backend /etc/init.d/apache_but start
Stop Apache Front & Backend /etc/init.d/apache_but stop
Restart Apache Front & Backend /etc/init.d/apache_but restart
