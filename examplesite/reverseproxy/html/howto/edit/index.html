Edit Apache Configuration

Please use the Edit Apache option in the Hacking-Lab menu for editing the Frontend Apache or Backend Apache web server.

You need to restart your Apache to make changes active!

Command Line

Item Command
Edit Frontend Apache mousepad /opt/applic/httpd/conf/httpd.conf
Edit Backend Apache mousepad /opt/applic/httpd/conf/httpd_backend.conf
Edit SSL/TLS Configuration mousepad /opt/applic/httpd/conf/extra/httpd-ssl.conf
Edit Pre-Auth Configuration mousepad /opt/applic/httpd/conf/extra/httpd-preauth.conf
Edit Web Application Firewall mousepad /opt/applic/httpd/conf/extra/mod_security.conf
Edit Strict Transport Security mousepad /opt/applic/httpd/conf/extra/httpd-hsts.conf
Edit Forensic Readiness mousepad /opt/applic/httpd/conf/extra/httpd-forensic-readiness.conf
Edit Content Security Policy mousepad /opt/applic/httpd/conf/extra/httpd-csp.conf
Edit Login Application mousepad /opt/applic/httpd/htdocs/login/login.php
