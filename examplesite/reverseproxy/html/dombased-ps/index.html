What we do

We do everything related to cyber security training and hands-on security practice labs.

Security Challenges

Train your brain with jeopardy style puzzles and riddles. Brainteaser improve your cyber security skills.

Crypto Puzzles

Ceasar, Rot, XOR, Diffie-Hellman and more security challenges are waiting for you. Crypto is one of the foundation of Hacking-Labs challenges.

Network Security

DNS hacking, hijacking, ARP spoofing and Layer2 security belong to this type of puzzle. Knowledge every pentester must have.

Web Hacking

Web is building the core of todays internet and e-services world-wide. A must do training for everyone.

Reverse Engineering

Important subject when it comes to understanding modern malware and APT (advanced persistent threats). Dig into the deep of ASM.

Mobile Security

The web is shifting from legacy to mobile. Important step to take for any security expert, understanding mobile security.
